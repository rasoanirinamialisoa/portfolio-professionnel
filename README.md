# Portfolio React

Bienvenue sur mon portfolio personnel développé avec React.

Ce projet présente mon parcours, mes compétences, mes projets et mes coordonnées professionnelles.

## Aperçu

- Présentation personnelle
- Compétences techniques
- Expériences et formations
- Projets réalisés
- Contact

- JavaScript (ES6+)
- HTML5
- CSS3
- Tailwind CSS
- GitHub
- Netlify

## Installation

Cloner le dépôt :

```bash
git clone https://github.com/rasoanirinamialisoa/portfolio-professionnel
```

Accéder au projet :

```bash
cd portfolio
```

Installer les dépendances :

```bash
npm install
```

Lancer le serveur de développement :

```bash
npm run dev
```

ou

```bash
npm start
```

## Build de production

```bash
npm run build
```
